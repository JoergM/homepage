Inoffizielle Portierung von Java 6 Leopard und Tiger verfügbar

29. November 2007 von Ralph Bergmann in Allgemein, Java, Mac Fan Club
Schlagwörter: Java, Sun


Java wird nicht direkt von Sun auf den Mac portiert, sondern durch Apple. Das hat gute Gründe – vor allem sind es betriebssystemspezifische Erweiterungen, die Apple einbaut. Aktueller Stand ist, auch bei Leopard, Java 5. Einige Java-Entwickler zeigten sich enttäuscht, dass in Leopard Java nicht aktualisiert wurde.Wer als Java-Entwickler auf die aktuellste Version angewiesen ist, muss nicht länger eine virtuelle Maschine bemühen, sondern kann eine Vorabversion von Java 6 herunterladen. Dies ist eine inoffizielle Version, welche auf den freien Quelltexten von Sun basiert. Unterstützung von Apple oder Sun gibt es daher nicht, so dass sich diese Portierung nur für experimentierfreudige (oder verzweifelte) Programmierer empfiehlt. Java 6 ist für Mac OS X 10.4 und 10.5 geeignet.”Quelle: MacLife

MacBook mit Leopard über Bluethooth + HSDPA mit der Internet verbinden

1. November 2007 von Jan Lolling in Mac Fan Club

Leopard hat die Nertzwerkkonfiguration deutlich vereinfacht. Das gilt auch für die Einrichtung eines Mobilfunktelefones als Modem in das Internet. Dazu sind folgende Schritte erforderlich:

1. Einrichten des Mobiltelefones als Bluethooth-Gerät.

Dieser Schritt ist weitgehend identisch zu Tiger (es werden in Leopard weniger Fragen gestellt und sehr sinnvolle Voreinstellungen gemacht). Wichtiger Hinweis: An seinem Telefon sollte am zuerst einen Namen für Bluethooth-Gerät vergeben um es später in einer Liste der Geräte leicht wieder zufinden.

2. Konfiguration des Internetzuganges

Wir benötigen: einen Benutzername (bei T-Mobile ist der Benutzer: “t-mobile”)

Wir benötigen ein Kennwort (bei T-Mobile ist das Kennwort: “tm”)

Wir benötigen einen APN (Access point name) der heisst bei T-Mobile “internet.t-mobile”

CID (Context Information Device) kann man auf den voreingestellten Wert “1″ stehen lassen. Hier war bei Tiger noch die Angabe eines CID-Stringes erforderlich, ist beim Leo entfallen oder besser gesagt nun im Script enthalten.
