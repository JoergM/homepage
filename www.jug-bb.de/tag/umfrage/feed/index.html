Immer mehr Firmen und Entwickler fragen nach Java Unterstützung von mobilen Endgeräten, insbesondere von JavaFX für Android und iOS.

Wenn wir als Java Community die Unterstützung von JavaFX für Tablets & Mobilgeräte als wichtig betrachten, dann sollten wir dies auch bekräftigen und mit der Teilnahme an der Umfrage ausdrücken.

Zur Umfrage geht es unter http://fxexperience.com/2012/12/javafx-for-tablets-mobile/

Die Java Usergroup Berlin-Brandenburg würde sich über eine rege Teilnahme sehr freuen.
