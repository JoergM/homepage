Der Vortrag:

JUnit ist der Klassiker für Unit-Tests mit Java. Mit TestNG steht ein alternatives leistungsfähiges Unit-Tests-Framework zur Verfügung, das in diesem Vortrag vorgestellt werden soll. Im Vortrag wird den Zuhörern das Rüstzeug für eingene Tests mit TestNG vermittelt.

Der Redner:

Oliver Fischer arbeitet als Projektmanager und Software Engineer in Berlin. Seine beruflichen Interessen gelten vorrangig dem Software Engineerung, dem er sich in seinem Blog unter www.swe-blog.net widmet. Zusätzlich ist immer auf der Suche nach effizienten Tools für seine Arbeit.
