Januar 2012 lädt die Java Usergroup Berlin-Brandenburg zusammen mit der adesso AG zum Vortrag Amazon Webservices for Java Developers mit Jan Borch von Amazon AWS ein.

Der Vortrag

Amazon Webservices for Java Developers – Part 1 (Jan Borch): Beanstalk, Cloud Formation, EC2, ElasticCache, SimpleDB, RDS, ELB, and more. Overview on how to leverage AWS services for Continuous Integration/Deployment/Optimization

Amazon Webservices for Java Developers – Part 2 (Jan Borch): Overview of the AWS tools for Java developer : Java AWS SDK, AWS eclipse plugin, and more. Bootstrapping Application with CloudFormation and integration with Chef and Puppet.

Der Redner

Jan Borch: Jan Borch ist Solution Architect im Amazon AWS EMEA Solution Architect team.

Der Treffpunkt

GLS Campus, Haus 3, Aula, Kastanienallee 82, 10435 Berlin (Prenzlauer Berg)

Hinweis

Die Anfangsuhrzeit kann sich ggf. noch verändern, sollte aber bis 13. Januar feststehen.

17:00 Uhr Einlaß und Zeit für Networking
17:30 Uhr Vortragsbegin (ca. 2 bis 2,5 Stunden)

Werner Vogels ist der Chefarchitekt der Amazon Webservices, des Cloud-Angebots von Amazon. Für seine Leistungen ernannte Information Week ihn 2008 zum CIO/CTO of the Year.

Der Vortrag

In seinem auf Englisch gehaltenen Vortrag wird Werner Vogels Amazons Cloud Computing-Ansatz und –Services vorstellen. Anschließend wird Amazons neues Angebot für Java-Entwickler, Amazon Elastic Beanstalk vorgestellt.

Nach der Veranstaltung wird es die Gelegenheit geben, mit Werner Vogels persönlich ins Gespräch zu kommen.

Der Ablauf

18:30 Uhr – Einlass und Zeit zum Networking
19:00 Uhr – Vortrag und anschließende Diskussionsrunde
