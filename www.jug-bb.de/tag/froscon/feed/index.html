Erstmalig dieses Jahr wird es eine Java-Subkonferenz geben, auf die ich hiermit aufmerksam machen möchte

Hallo Java Freunde,

während der FrOSCon 2009 findet dieses Jahr exklusiv eine Subkonferenz zum Thema Java statt. Wir suchen interessante Beiträge rund um das Thema Java mit einem Bezug zu Open Source / freier Software. Jegliches Vortragsthema ist willkommen, hier nur ein paar Vorschläge:

  • Zukunft von Java und der JVM im Open Source Umfeld
  • Features von Java 7
  • OpenJDK
  • Da Vinci Machine Project
  • Android
  • Java Content Repository
  • Cloud Computing
  • Erfahrungsberichte aus Java Projekten mit Open Source Einsatz

Der Call for Papers ist ab sofort bis zum 23.05.2009 geöffnet. Weitere Informationen unter:
http://www.froscon.de/programm/call-for-papers.html

Da die FrOSCon eine ehrenamtlich organisierte Konferenz ist, die sich im Wesentlichen durch Sponsoring finanziert, freuen wir uns natürlich auch über finanzielle Hilfe sowie jegliche Unterstützung in anderer Form.

Wir freuen uns auf Einsendungen und eine spannende Java Konferenz am 22. und 23. August 2009 in Sankt Augustin.

Marc Guillemot und Stéphane Boulanger
