Java User Group Berlin Brandenburg » bed-con http://www.jug-bb.de Der Blog der JUG-BB Sun, 29 Dec 2013 15:02:50 +0000 de-DE hourly 1 http://wordpress.org/?v=3.8.2 Call for Papers für die Berlin Expert Days 2013 http://www.jug-bb.de/2012/11/call-for-papers-fur-die-berlin-expert-days-2013/ http://www.jug-bb.de/2012/11/call-for-papers-fur-die-berlin-expert-days-2013/#comments Sun, 25 Nov 2012 15:24:00 +0000 http://www.jug-bb.de/?p=969 Für die 2013 zum dritten Mal stattfindenden Berlin Expert Days hat der Call-For-Papers begonnen.

Die Berlin Expert Days bieten technisch-orientierten Entwicklern, Architekten und Administratoren eine Plattform zum Erfahrungsaustausch und persönlichem Gespräch. Die Themenbereiche drehen sich rund um die Java Welt, z.B.: Java, Java EE, Spring, JavaScript, funktionale Sprachen, NoSQL, Continuous Delivery, DevOps und User Experience.

Zu diesen und weiteren Themen können Vorträge unter http://bed-con.org/2013/call-for-paper/ bis zum 14. Januar 2013 eingereicht werden. Die Konferenz selber wird am 4. und 5. April 2013 in Berlin stattfinden.

Hinweis in eigener Sache: Die Java Usergroup Berlin-Brandenburg ist Partner und Unterstützer der Berlin Expert Days

Die BerlinExpertDays ist eine technisch-orientierte Konferenz und versteht sich als Plattform, durch die alle Interessierten zusammenkommen und
sich austauschen können. Um die Einhaltung dieser Zielstellung kümmert sich ein offenes Programm-Komitee.

Vorträge werden noch gesucht

Die Einreichung von Vorträgen hat begonnen. Das Programmkomitee freut
sich schon auf die vielen Vorschläge. Die Themenbereiche drehen sich
rund um die Java Welt, z.B:

  • Java
  • Java EE
  • Spring
  • JavaScript
  • funktionale Sprachen
  • NoSQL
  • Cloud Computing
  • User Experience

Der Call For Papers endet am 31. Januar 2012. Eigene Vorträge können über die Webseite der BerlinExpertDays unter http://bed-con.org/call-for-paper/ eingereicht werden.
